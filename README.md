# AppTemplateOfReactNative

## 技术栈
- react
- react-native
- redux
- immutable
- react-native-router-flux
- react-native第三方原生组件库

## 简单的目录说明
- src（源码文件）
	- common（工具库）
	- components（项目组件）
	- reducers（reducer）
	- routes（页面）
	- App.js（入口文件）
- tools（工具库，目前只有构建器）
- .eslintrc（eslint 规范）

## 命令
```
	npm start --- 开启服务
	npm run gen Home --- 构建器，一行命令，自动构建页面模版
	npm run test --- 开启测试
```
